# CSE3100 Final Project

A starter full-stack project for CSE 3100 final submission.

## What is included

- Frontend (HTML/CSS/JS): login, register, dashboard, todo list
- Backend (Node/Express): auth routes, user routes, JWT auth
- Database: MySQL queries and the course final project PDF (in /database)
- README and package.json

## How to use

1. Import `database/queries.sql` into your MySQL.
2. Update `backend/config/database.js` with your DB credentials.
3. Install dependencies: `npm install`
4. Start the server: `npm run dev` (if you have nodemon) or `npm start`
5. Open `frontend/index.html` in your browser (or serve using a static server)
